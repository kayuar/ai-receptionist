# System Overview

AI Receptionist consists of the following components:

1. Telephony Layer
Handles inbound and outbound phone calls.

2. Speech-to-Text
Converts caller voice to text.

3. Conversation Engine
LLM-powered reasoning engine.

4. Tool Execution Layer
Triggers workflows such as booking appointments.

5. Integrations
Calendars, CRMs, and backend systems.

6. Text-to-Speech
Converts AI response back into voice.
