# AI Receptionist

An open-source voice AI receptionist platform that helps businesses automate inbound phone calls using conversational AI.

The system allows businesses to deploy AI-powered voice agents capable of answering calls, booking appointments, answering FAQs, and routing conversations to the right departments.

## Vision

Most businesses still rely on human receptionists to handle repetitive phone interactions.

The goal of this project is to build an open and extensible AI platform that can act as a reliable first point of contact for businesses.

Instead of building closed SaaS tools, this project aims to provide an open framework that developers can extend and customize.

## Core Capabilities

- AI-powered voice conversations
- Appointment booking
- FAQ handling
- Call routing
- CRM integrations
- Calendar integrations
- Business workflow automation
